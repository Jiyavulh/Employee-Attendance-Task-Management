# Employee Attendance and Task Management System

## 📌 Description
A web-based system to manage employee attendance and tasks using Spring MVC, Hibernate (JPA), and MySQL.

## 🚀 Features
- Employee Registration & Login
- Attendance Tracking
- Task Creation & Assignment
- Task Status Update
- View Attendance Records

## 🛠 Tech Stack
- Java
- Spring MVC
- Hibernate (JPA)
- MySQL
- JSP / HTML
- Maven
- Tomcat
